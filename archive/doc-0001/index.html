    Документ — официальная ранжированная таблица результатов грантового конкурса «Konkurs Grantowy na rzecz Białorusi» за 2023 год. Конкурс проводила Fundacja Solidarności Międzynarodowej (FSM) — польская государственная фундация, действующая по поручению МИД Польши (Ministerstwo Spraw Zagranicznych RP) в рамках программы «Wsparcie Demokracji» и пакета помощи «Solidarni z Białorusią». Три списка: получившие финансирование (5 организаций, суммарно 2 073 460 PLN), отклонённые по баллам (8 организаций), отклонённые по формальным причинам (10 организаций). Для каждого проекта указаны заявитель, название проекта, приоритеты и количество баллов; для победителей — также сумма финансирования и страны реализации.

    Документ опознаётся по фирменному графическому знаку FSM (стилизованные фигуры в национальных цветах) в верхнем колонтитуле каждой страницы.

    Аналитическая ценность. Документ позволяет рассмотреть приоритеты польского государства в распределении грантовой поддержки беларуской эмиграции. На основании самих данных, без интерпретативных скачков, фиксируются следующие закономерности:

    1. Концентрация бюджета. 980 000 PLN из 2 073 460 PLN (47% от суммы победителей) выделены одной организации — Fundacja Białoruś Przyszłości — на проект «разработки дорожной карты защиты основных прав». Это аналитико-правовая, а не операционная работа.

    2. Профиль победителей. Из пяти получивших финансирование: один аналитический хаб (Białoruś Przyszłości, 980 000), два медийных проекта (Strefa Solidarności / Белсат — 300 000; Biuro Białoruś w Fokusie — 400 000), одна общеобразовательная программа (Kolegium Europy Wschodniej — 142 000) и одна организация с заявленной миссией «развитие демократии» (BYPOL — 251 460). Прямой помощи беларусам в Польше (юридическая, психологическая, адаптационная) среди победителей нет.

    3. Профиль отклонённых по баллам. Восемь организаций, чьи проекты были направлены на конкретные группы и операционную поддержку: репрессированные учителя (Edukacja dla Demokracji, 14,0), культура в эмиграции (Willa Decjusza, 12,5), поддержка беларуской общины в центральной и западной Польше (HumanDoc, 12,0), образовательная программа (Polska360, 10,5), сеть НКО и адаптационно-интеграционных центров (Linking Media, 4,5), «Wybieramy!» (Fundacja Życie, 4,5), профессиональная опека и непосредственная помощь репрессированным (Wspólnota Polska, 2,0), трудоустройство беженцев (Sustainable Strategies Network, 1,0).

    4. Аномалия с Wspólnota Polska. Stowarzyszenie «Wspólnota Polska» — одна из старейших и крупнейших польских организаций по работе с диаспорой, существующая с 1990 года, с инфраструктурой и многолетним опытом — получила 2,0 балла за проект прямой помощи репрессированным. Низкая оценка для опытной организации с заявленной операционной (а не риторической) миссией.

    5. Профиль отклонённых формально. В этом списке также преобладают практические инициативы: Centrum Nowaja Ziamlia в Белостоке (один из реальных центров для беларусов в эмиграции), психологическая помощь беженцам (Czabor), спортивный медиапроект Tribuna.com, школа лидеров изменений (BYPROSVET), резурс памяти Дануты Хлусни (International Freedom and Democracy Foundation), форум беларуского бизнеса за рубежом, проект «беларуского самоуправления будущего» (Partnerstwo Międzynarodowe).

    Содержательный паттерн. Если систематизировать тематику победителей и отклонённых, на данных видна следующая закономерность: финансирование концентрируется в проектах, чья основная аудитория — международное сообщество (медиа на эмиграцию, аналитические доклады, дорожные карты для западных институтов), а не сами беларусы в Польше. Проекты прямой помощи (психологической, юридической, образовательной, трудоустройства) системно отклоняются — как по баллам, так и формально. Это требует дальнейшей проверки на нескольких годах конкурса; пока зафиксировано как наблюдение по данным за 2023 год.

    Ограничения данных. Документ не содержит: подписей членов оценочной комиссии (нужны как отдельные документы), детализации критериев, по которым выставлялись баллы, информации о возможной апелляционной процедуре, отчётов о фактической реализации профинансированных проектов. Все эти данные предстоит привязать к карточке отдельными doc-XXXX по мере поиска.
